# sawa : rest api base package
=========================

dependencies

    dropwizard:1.0.7
    postgresql:9.1
    jetty:9.3
    twilio:7.0

you can package by yourself

    mvn package
    
or use maven dependency in pom.xml

    <dependency>
        <groupId>com.github.shouaya</groupId>
        <artifactId>sawa</artifactId>
        <version>0.0.1</version>
    </dependency>
